# 💻 **_Zeus_** 

**_Zeus_** é o frontend do portfólio pessoal integrado ao backend [**_Prometheus_**](https://github.com/NukeZzZin/prometheus) Desenvolvido em **_NodeJS_** com **_React (Vite)_**, adota práticas modernas de desenvolvimento web, utilizando **_Material UI_** para interfaces acessíveis, **_Zustand_** para gerenciamento de estado simples e **_Axios_** para comunicação eficiente com APIs.

## 🌎 **_Prerequisites_**

- Instalar [**_Git_**](https://git-scm.com/).
- Instalar [**_NodeJS_**](https://nodejs.org/).
- Instalar [**_Yarn_**](https://yarnpkg.com/).

### 🚚 **_Installation_**

1. Clone o repositório usando **_`git clone https://github.com/NukeZzZin/zeus.git`_** no seu terminal.
2. Entre na pasta do repositório usando **_`cd zeus`_** no seu terminal.

#### 🐱‍💻 **_Local developing run_**

1. Faça toda a preparação do **_Zeus_** usando **_`yarn install`_** no seu terminal.
2. Faça toda a compilação do **_Zeus_** usando **_`yarn build`_**
3. Inicialize o **_Zeus_** usando **_`yarn preview`_** no seu terminal.

#### 🏎️ **_Local production run_**

1. Faça toda a preparação do **_Zeus_** usando **_`yarn install`_** no seu terminal.
2. Inicialize o **_Zeus_** usando **_`yarn dev`_** no seu terminal.

## 🔗 **_Integração com Prometheus_**

O Zeus se comunica com o backend **Prometheus** via API REST. Certifique-se de que o backend esteja rodando e que a variável `VITE_API_URL` no `.env` do Zeus aponte corretamente para o endereço do Prometheus.
